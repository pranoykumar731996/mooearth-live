## How to Record the Final Video

### Method 1: OBS Studio (Recommended)
1. Open `http://localhost:3000/promo/` in Chrome
2. Set Chrome to full-screen (F11)
3. In OBS: Add "Window Capture" → Select Chrome
4. Set output to 1920×1080, 60fps, CRF 18 (high quality)
5. Click "Start Recording"
6. Wait for the 30-second auto-play to complete
7. Stop recording → Export as MP4

### Method 2: Browser Extension
1. Use "Screen Recorder" Chrome extension
2. Record the tab playing the presentation

### Method 3: Windows Game Bar
1. Press Win+G
2. Click Record
3. Captures the full window

### Post-Production (for voice-over and music)
1. Import the screen recording into **CapCut** / **DaVinci Resolve** / **Premiere Pro**
2. Add the voice-over audio track (record using the script above)
3. Add background music track
4. Sync audio to visual cues using the timestamp table
5. Add final color grading if desired
6. Export in multiple formats:
   - YouTube: 1920×1080, H.264, 15-20 Mbps
   - Instagram/TikTok: Crop to 1080×1920 (vertical)
   - Twitter/X: 1920×1080, max 2:20
   - LinkedIn: 1920×1080
